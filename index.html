Building the Translation Key: From Light Absorption to Concentration

Every UV‑Vis spectrum gives absorbance vs wavelength. We translate absorbance into real concentrations by building a robust calibration curve.

Peak detected near 671 nm y = 0.10944x + 0.03158 R² = 0.9964

The Real Scientific Process

  1. Prepare reference standards: known MB concentrations (0.5, 1, 2, 5, 10, 15 ppm)
  2. UV‑Vis measurement: full absorbance vs wavelength for each standard
  3. Peak selection: automatically focus on the maximum (≈671 nm)
  4. Linear relationship: fit concentration vs absorbance
  5. Validation: R² ≥ 0.995 for reliable predictions

From UV-Vis Spectra to Concentration Profiles

For every time point in our experiment (0, 30, 60, 90, 120 minutes), we get a UV‑Vis spectrum. System instantly converts each spectrum into a concentration value, building the degradation curve in real‑time.

What Actually Happens:

  1. Time-Series UV-Vis Data: At each time point (0, 30, 60, 90, 120 min), take UV‑Vis spectrum
  2. Peak Absorbance Extraction: System finds the maximum absorbance value from each spectrum (same wavelength used in calibration)
  3. Calibration Application: Apply the calibration equation to convert absorbance to concentration
  4. Concentration vs Time Plot: Build the degradation curve automatically
